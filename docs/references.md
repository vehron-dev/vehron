# References

## Purpose

This page collects research references that inform VEHRON models and
documentation. It is not a claim that every implementation is a paper-exact
reproduction. Each model page should still state whether the implementation is:

- literature-faithful
- literature-inspired
- or an engineering approximation

## Cabin Thermal Modeling

1. Marcos, D., Pino, F. J., Bordons, C., and Guerra, J. J., 2014
   "The development and validation of a thermal model for the cabin of a vehicle"
   Applied Thermal Engineering 66, 646-656
   DOI: https://doi.org/10.1016/j.applthermaleng.2014.02.054

2. Torregrosa-Jaime, B., Bjurling, F., Corberan, J. M., and Di Sciullo, F., 2015
   "Transient thermal model of a vehicle's cabin validated under variable ambient conditions"
   Applied Thermal Engineering 75, 45-53
   DOI: https://doi.org/10.1016/j.applthermaleng.2014.05.074

3. Marshall, G. J., Mahony, C. P., Rhodes, M. J., Daniewicz, S. R., Tsolas, N., and Thompson, S. M., 2019
   "Thermal Management of Vehicle Cabins, External Surfaces, and Onboard Electronics: An Overview"
   Engineering 5(5), 954-969
   DOI: https://doi.org/10.1016/j.eng.2019.02.009

4. Zhao, L., Zhou, Q., and Wang, Z., 2024
   "A systematic review on modelling the thermal environment of vehicle cabins"
   Applied Thermal Engineering 257, 124142
   DOI: https://doi.org/10.1016/j.applthermaleng.2024.124142

## Usage Rule

When a VEHRON model has a research basis, the relevant model page should:

- point to one or more entries from this page
- state whether the implementation is exact or simplified
- document any known deviations from the cited model family
