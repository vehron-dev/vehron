# VEHRON Contributor License Agreement

This document describes the contributor terms for VEHRON.

It is intended to keep copyright and relicensing rights clear as the project
grows. It is a lightweight project document written for practical clarity, not legal advice.

## Purpose

VEHRON is being developed as an open-source project today, with the possibility
of future dual licensing or commercial relicensing. To make that possible,
contributors must grant the project maintainer broad rights to use contributed
material.

This structure is intentional. VEHRON is independent work rather than institution-sponsored work, and the goal is to keep the project open while also avoiding governance dead-ends later.

The intention here is not to diminish contributor value. It is to be transparent about how the project is being stewarded from the beginning.
